# arcmodel

Model management and routing abstractions for Arc.

## Layer position

arcmodel is a scaffolding package. No other Arc package currently depends on it.

## What it provides

Status: scaffolding only — no public API yet. The `__init__.py` contains only `__version__ = "0.0.1"`.

Future scope: model selection, routing, capability discovery, and tiered model management for multi-provider deployments.
